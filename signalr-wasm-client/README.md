# signalr-wasm-client

This is a simple Rust-based WASM client for SignalR, the protocol by Microsoft.

## Contents:

- `SignalRServer`: A reference implementation of a SignalR server using the official Microsoft C# packages.
- `async-test`: me trying to understand Rust async
- `js_client`: A reference implementation of a SignalR client using the official Microsoft TypeScript packages
- `signalr-client`: The client as a `pnpm` app with:
  - `app`: A basic chat web application
  - `signalr-wasm`: The actual Rust client
